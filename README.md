# PhotoCamper

## サイト概要
campで撮った写真・自身のギアの紹介・キャンプ場の紹介・料理のレシピを投稿するサイトです！

### サイトテーマ
campの良さを共有・紹介して行こう！！

### テーマを選んだ理由
campが好きなので、campが好きな人たちと写真や料理のレシピ・ギアを共有・紹介できるサイトを作りたいと考えたから！

### ターゲットユーザ
全国のcamper

### 主な利用シーン
・campギアの投稿・閲覧<br>
・camp飯の投稿・閲覧<br>
・camp場の投稿・閲覧
